# simple-restapi-go



### How to use

Using MongoDB:

```bash
export URL_DB=mongo
export MONGO_URL=<host>
export MONGO_TIMEOUT=30
export MONGO_DB=<database>
```

Using Redis:

```bash
export URL_DB=redis
export REDIS_URL=redis://localhost:6379
```



### Running Application

```bash
go run main.go
```
